PLATAFORMA JB EDITABLE 1.0

Esta versión incluye un panel de administración dentro de la web.

IMPORTANTE:
- El panel guarda cambios en el navegador usando localStorage.
- Sirve para probar y editar visualmente sin tocar código.
- Para que los cambios queden públicos para todos, hay que exportar la configuración y reemplazar config.js.

CÓMO PUBLICAR EN CLOUDFLARE:
1. Descomprimir el ZIP.
2. Subir el contenido de la carpeta a Cloudflare Pages.
3. Verificar que index.html esté en la raíz.

CÓMO EDITAR:
1. Abrir la web publicada.
2. Ir a la sección "Panel".
3. Cambiar textos, links, imágenes o PDF.
4. Guardar.
5. Usar "Exportar configuración" cuando quieras dejar esos cambios como versión definitiva.

PRÓXIMA EVOLUCIÓN:
- Panel de administración real con base de datos.
- Login de clientes.
- IA real con OpenAI.
- Pagos conectados.
